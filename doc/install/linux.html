Linux Installation

What follows is a detailed step-by-step guide on installing Azul3D on a Linux 386/amd64 platform. For other platforms please see the installation page.

☣  Install Go ☣  Dependencies ☣  Ubuntu 14.04 ☣  Ubuntu 13.10 ☣  Arch Linux ☣  Compiling The Examples

» Install Go (top)

Install the latest version of Go from here.

Note: Azul3D requires at least Go version 1.3

Set the environment variable GOPATH, for example:

☣  /home/joe/godev

» Dependencies (top)

☣  Go 1.3 ☣  git (to clone the source code). ☣  gcc ☣  OpenGL development files ☣  libx11 development files ☣  libx11-xcb development files ☣  libxcb development files (including icccm, image, randr, render-util, and xcb-xkb parts) ☣  libfreetype, libbz2 and libzip static libraries and headers

Single-line installers for dependencies are provided below. If your distro is not listed here please help us by creating an issue.

» Ubuntu 14.04 (top)

Using apt-get on Ubuntu 14.04 you can install all of the dependencies by running:

☣  sudo apt-get install build-essential git mesa-common-dev libx11-dev libx11-xcb-dev libxcb-icccm4-dev libxcb-image0-dev libxcb-randr0-dev libxcb-render-util0-dev libxcb-xkb-dev libfreetype6-dev libbz2-dev

» Ubuntu 13.10 (top)

With Ubuntu 13.10, you need to add the xorg-edgers PPA for the libxkb-xcb-dev package:

☣  sudo add-apt-repository ppa:xorg-edgers/ppa ☣  sudo apt-get update

Then you can use apt-get to install everything:

☣  sudo apt-get install build-essential git mesa-common-dev libx11-dev libx11-xcb-dev libxcb-icccm4-dev libxcb-image0-dev libxcb-randr0-dev libxcb-render-util0-dev libxcb-xkb-dev libfreetype6-dev libbz2-dev

» Arch Linux (top)

Using pacman on Arch Linux you can install all of the dependencies by running:

☣  pacman -Sy base-devel git mesa libx11 libxcb xcb-util-wm xcb-util-image libxrandr xcb-util-renderutil libxkbcommon-x11 freetype2 bzip2

» Compiling The Examples (top)

Azul3D is installed via the standard go tool. To download and compile the example source code run:

☣  go get azul3d.org/examples.v1 ☣  go get azul3d.org/examples.v1/...

The example binaries will be placed into the $GOPATH/bin directory.
