OS X Installation

What follows is a detailed step-by-step guide on installing Azul3D on a Mac OS X platform. For other platforms please see the installation page.

☣  Install Go ☣  Dependencies ☣  Compiling The Examples

» Install Go (top)

Install the latest version of Go from here.

Note: Azul3D requires at least Go version 1.3

Set the environment variable GOPATH, for example:

☣  /home/joe/godev

» Dependencies (top)

Xcode must be installed from the App Store.

Homebrew is used to install the dependencies. To install homebrew run:

☣  ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

Then install the FreeType dependency:

☣  brew install freetype

» Compiling The Examples (top)

Azul3D is installed via the standard go tool. To download and compile the example source code run:

☣  go get azul3d.org/examples.v1 ☣  go get azul3d.org/examples.v1/...

The example binaries will be placed into the $GOPATH/bin directory.
